Continuous Delivery of iOS Apps

The book connects iOS development with DevOps principles.

Everything as code is a DevOps principle where the infrastructure, build environment, build and distribution are treated as application code just like the app: Versioned and accessable by every devloper

The book will bring you closer to these principles and show you how to use them in the development of iOS apps. You will benefit whether you develop alone or in a team.

Chapters

The book contains following chapters

  • Explanation of Continuous Integration, Delivery & Deployment
  • Build Server Provisioning & CI Cloud Service
  • Xcode Build Process
  • fastlane
  • iOS Project Configuration & Build Artefacts
  • Dependency Management & Installation
  • Project Setup
  • Certificate & Provisioning Profile Managment
  • Quality Checks
  • Distribution of Build Artefacts
  • Crash Reporting & Error Logging
  • Project and Pipeline Documentation

Learning by doing

The explanations in the book are linked to executable code examples.

Independent chapters

The chapters will be independent from each other in order to implement only parts of the DevOps principles.

Source code available

The entire source code is provide via GitHub. You only need to fork and run it.
